### 购买方法
如[官网](http://fujirock-eng.com/ticket.html#purchaseticket_overseas)所示，非日本居民购票渠道包括 [GAN-BAN(FRF官方商店)](http://ganban.net/?p=30564), [e+](http://eplus.jp/ib-frf17), [iFlyer](https://iflyer.tv/fujirock2017/), 以及我们这次光顾的 [TICKETFLAP](https://www.ticketflap.com/frf17)。
<br/>
选择 Ticketflap 最直接的好处是有网站有繁体中文，但注意它只提供三日通票和营地票。支持银联卡及信用卡支付（不能使用支付宝）。填写真实的邮箱及护照上的英文姓名，支付后你会收到一封邮件，里面有门票的下载链接，打印出来带到现场，找到 TicketFlap 的换票点`点击跳出互动地图`，出示门票和护照（或其他身份证明），就可以换到三日手绳（如果购买了营地票还有营地手绳），用来出示给现场工作人员看的。
<br/>
### HINTS
手绳才是真正的「门票」。有了手绳，你便可以自由出入音乐节（比如到湯沢町镇上的便利店买东西再回到音乐节），注意手绳一旦戴上，不用剪刀是拆不下来的，拆下来也就戴不回去了，建议不要戴得太紧。

## 天气及衣物
FRF 在每年7月的最后一个周末举行，虽然正值炎夏，但湯沢町海拔800m，苗场本身是个滑雪场，从山脚到山顶也有约200m，加上温带海洋气候，昼夜温差比较明显，早上太阳高升后，气温大约25～28˚C，傍晚到凌晨约20˚C，凌晨3点直到清晨就只有15˚C左右了。

所以，白天天气好的时候可以短装出行，晚上必须准备长袖甚至秋冬装防寒。尤其是如果你打算露营，最好准备睡袋或者被子。

凌晨到早上、甚至到下午都很有可能会下雨，雨具是必不可少的。建议随身带雨衣，在这样人员密集且经常需要走动的场地，打伞极其不方便，而且看演出的时候，是不允许打伞的。

音乐节场地内并没有柏油路，只有少数用木头搭的栈道，其余都是泥路，下起雨来，人走得多了，自然泥泞不堪，所以千万不要装帅穿球鞋，绝大多数人的选择是雨鞋或凉鞋。雨鞋当然是天不怕地不怕有路的地方都可踏，但假如你脚上爱出汗，在雨鞋里面闷一天也是很难受的。我们的选择是登山凉鞋`此处配图`，适合长时间徒步，脏了的话用水冲洗即可（但可供洗脚的地方并不多）。

### HINTS
- 官方纪念品店有很好看的雨衣出售，不妨到了现场再买，但购买的人很多，必须提早才买得到，不然还是自带吧。全场纪念品店有三个`此处互动地图`。
- 除了官方纪念品店，这次 Tower Record 的商店`此处互动地图`也出售他们与KIU合作的雨衣+渔夫帽套装，我们就是在他家买的。

## 露营
作为野外音乐节，FRF 与 Coachella 和 Glastonbury 一样，露营是音乐节的一大乐趣。如果你完全没有露营经验（像我们这样），或者没有试过在音乐节露营，在决定是否露营前，可以先根据下面几点，评估一下露营享受到的乐趣，是否值得作出某些牺牲：

Pros:
- 费用低廉。购买营地票只需 JPY 3,000(~RMB 184)，你便可以最长在帐篷里呆4个晚上（周四、五、六、日）相比起旅馆或民宿，简直是像免费一样。
- 不用担心每日来回车程。如果你不是住在湯沢町（苗场滑雪场即FRF所在地的小镇）的旅馆，你需要每天坐穿梭巴士来回（穿梭巴士到越后汤泽站每日每人的车费是 JPY 500)，而且穿梭巴士晚上在10:30左右结束营业，就是说，你不能把当天的压轴演出看完（差不多是1:00）就得走了。不要想能叫到Uber，没有的。
- 完整的野外音乐节体验。

Cons:
- 需要很多装备。下一部分会详细列出必须的装备清单。
- 睡眠质量因人而异。营地里总有其他人在深夜玩到两三点还不睡的，偶尔也有人大声聊天。到早上也有很多人早起，如果你不是其中一员，就会听到你帐篷外人来人往。另外，温差也会影响你的睡眠，凌晨的时候很冷，到太阳出来的时候你又会觉得帐篷里闷热。

装备:
[参考](https://www.rei.com/learn/expert-advice/festival-camping-checklist.html)
- 优质帐篷。防潮、通风是最低要求。面积最好能够睡之外还能有地方放行李的，因为会下雨，如果行李本身不防水，放在户外（没有听说过有人在营地被偷东西的事情，但不保证）就很麻烦了。如果加上一个遮阳篷能够让你跟朋友们坐在户外聊天烧水泡面煮咖啡欣赏仙境般的山色就相当完美了。
> 帐篷租赁。FRF官方合作伙伴 [GAN-BAN](http://ganban.net/?p=30664) 提供帐篷出租服务，先到先得。出租的帐篷为 [Captain Stag](http://www.captainstag.net) (日本本土品牌) 的一款四人帐篷以及一张垫子，租金 JPY 20,000 (~RMB 1,217)/4日，押金JPY 5,000，在网上付全款预订之后凭邮件和身份证明，在指定时间内（周四12:00-24:00, 周五、六、日9:00-24:00，周一6:00-11:00）到 GAN-BAN 的摊位领取。注意需要保持帐篷内干燥、干净，不要穿鞋进帐篷。
- 睡袋。营地不是平地而是在山腰的几片草地上，而帐篷的底面非常薄，一个舒适的睡袋至少可以让你躺下不觉得疼。而且到了晚上寒冷的时候还可以保暖。
- 防潮垫。它是一张铺在帐篷底下（不是帐篷里面）的尼龙薄布，买帐篷一般不配有防潮垫，草地、晨露、雨天都让防潮垫成为必需品。
- 床垫。如果睡袋还不足以让你感到舒服，可以考虑加（换）配床垫。建议选择充气床垫，因为收起来体积比较小。考虑到行李的总体积/重量，而睡袋收纳的体积也不小，一般搭配应该是「床垫+被子」或只用「睡袋」。
- 枕头。同上建议充气枕头。或购买带有枕头的睡袋。
- 塑胶锤子。帐篷的四个角和四条边会有一条尼龙绳用以固定，绳子的末端需要用金属勾钉打进地下，这时候你需要一个锤子。塑胶锤子因为轻便，成为大部分人的首选。你可以向周围的营友借，我们也是这样做的，但因为锤子本身有损耗，现在回想起来也怪不好意思的。你也可以在入口的「露营帮助中心」租到锤子，租金不确定。
- 光源。营地是没有灯的。你可以选择一般的露营灯或者电筒。我们的选择是电筒，既可以晚上到处耍的时候用来找路，在帐篷里面也可以固定在顶部充当光源。
- 电源。一个20000毫安的充电宝是最好的电源。抱歉我们没有找到有公共充电的地方，不过这么多人，就算有也救不了你。实际上，四天内，2w毫安一个人根本用不完，要充电的设备包括手机、电筒、相机、GoPro，离开营地的时候充电宝余量显示为46%。
- 洗漱用品。走出营地不远处便有免费公共浴室全天候开放，面向浴室右边有洗手盆用以刷牙洗脸，再走过去还有JPY 500/人次的澡堂，所以保持个人清洁完全不是问题。高峰时段（上午和晚上）男生浴室大约会有15人排队，女生浴室排队的人多一点，大约20～30人。所以注意错峰洗澡。
- 垃圾袋。在日本，乱丢垃圾是非常奇葩的行为，请勿轻易挑战社会的底线。最好是准备好几个垃圾袋以便分类，如何分类在另一部分有详述。
- 衣物。FRF场内是没有洗衣服的地方的，所
- 以要带够这么多天的衣服。我们建议带上一次性纸内裤。
- 纸巾、湿纸巾及消毒液。
- 急救用品。

非必需但绝对锦上添花的装备：煤气炉、烧水壶、烧烤架、冷藏箱、太阳镜、羽绒服等等。

### HINTS
- 营地开放时间是周四中午12:00。
- 如果是全女生团，你可以选择离营地出入口更近的女生专用区域。
- 越早到达营地，你能占到的位置越好。扎营的地方尽量选择平地，一来躺着舒服一点，而来时间久了帐篷也不会变形。我们是下午3点左右到达营地的，当时所有平地都已经被占据了，只好找了一处稍微有点坡度的地方，最后走得时候，由于床垫及人的重力，帐篷的下方已经鼓起来了。
- 选择位置有点小技巧：睡觉的地方最好不要贴着通道，人来人往非常热闹的。也避免在草地中心，迟早你附近会塞满了其他人的帐篷，这时候出入帐篷比较麻烦。

## 旅馆

因为露营，我们这次没有住旅馆的体验。但我们看到，湯沢町上有很多旅馆和民宿，也许是因为这里本来就是度假的地方吧。大部分旅馆和民宿都是在FRF现场外走约十分钟的路程就有，也非常方便。另外，在越后汤泽站附近也有大量的酒店、旅馆，因为当地是个温泉胜地，只是那里去苗场需要40分钟左右的车程，穿梭巴士是 JPY 500/人 载你到苗场，从苗场返回越后汤泽免费。FRF这几天想必是旅游高峰，酒店费用可能水涨船高，建议提早预订。

## 饮食

### 苗场篇
在FRF耍是不需要担心饿肚子的，前提是要带够现金。整个FRF区域（包括FRF舞台区域和场外）共有12个地方有餐饮小店，它们不是餐厅，而是类似「食街」的在各个区域边上一字排开的摊位，所以没有座位，排队、付钱然后拿走，站着吃、坐在自带的椅子上吃或者坐在地上吃请随意，但切记：排队！以及不要乱扔垃圾！「食街」的打烊时间不确定，但至少我们醒着的时候（直到凌晨两三点）它们都是营业的。

食物的品种极其丰富，日本料理（拉面、乌冬、关东煮、串烧、咖喱）、西式简餐（热狗、三文治、沙拉、）、披萨、意面、印度料理、泰国料理等等，饮品方面也是花样繁多：扎啤几乎每个摊位都会提供，毕竟是音乐节，微醺是最好的状态，另外还有精酿啤酒、鸡尾酒、清酒、烧酌、咖啡、牛乳品、软饮等等。虽然选择很多，但品质参差不齐，我们吃过很难吃的乌冬，也吃过很好吃的咖喱吉列牛排饭。价格方面，各品种都会比东京市区同类贵大约 JPY 100，啤酒是600～700，面饭类是500～700。

12个餐饮区域这里就不一一详述了，我们的互动地图里面会有标示，这里介绍几个我们印象比较深刻的：
- 场外。即穿梭巴士下车处往苗场方向走5～8分钟的接待区域，这里除了有「食街」，还有最大的纪念品商店、手绳换领处等。非常合适长途旅行到达后慰藉饥肠辘辘的你。
- Oasis。场内最大的休闲区，摊位最多。周四晚上的「前夜祭」也在这里发生。有好些大品牌也在这里设置了摊位，比如 Mevis、Sony、Barcadi 等等，还时不时有杂技魔术表演，非常欢乐。红舞台（Red Marquee）也在这旁边。
- Orange Cafe。最远端的场地，有烧烤场，可以在这里买生肉自己烤，有座位。
- Day Dreaming & Silent Breeze。全场制高点，需要坐缆车上去，上边有一家很大的西餐厅，可惜我们上去的时间太晚（下午5点！），已经停止接待了。

### 汤泽町篇
苗场外便是汤泽町小镇，镇上有很多日式料理店、居酒屋等，价格稍贵，我们经过的时候看了下菜牌，一道菜四位数的价格让我们望而却步。四天里我们只光顾了一家拉面店，只有一位老伯伯在经营，不过汤咸了点。镇上还有旅游胜地必备之特产店，新泻盛产大米，所以用大米做的各式零食（糍粑、米饼等）、清酒琳琅满目。
	镇上有一家便利店，多亏了它，拯救了囊中羞涩的我们。我们自然很想尝遍场内美食，但现金不足以挥霍，有好多次都是靠杯面解决了温饱问题，好在日本的杯面是真的比国内好吃且选择丰富。另外冲泡的味增汤配微波炉白米饭也相当不错哦！便利店还能买到日常饮用水（总不能每时每刻都在喝酒吧）和便宜的易拉罐酒精饮品。注意在场外购买的酒精饮品按规定是不能带进场内的哦，入场时会（不那么严格地）检查你的背囊。
